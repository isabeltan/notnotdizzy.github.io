Lazuli

ux/ui

Lazuli is a mobile app for a fictional clothing brand, which I created for my Visual Interation Design class. The objective of this project was to design an app for tracking customer loyalty. I chose a minimal and monochromatic interface to allow product items to shine and considered a system which would allow users to navigate screens efficiently. The result is a clean and easily navigable interface which customers can use to purchase, favourite, and curate lists of products to accumulate points.
